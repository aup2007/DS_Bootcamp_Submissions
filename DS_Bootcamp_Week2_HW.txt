1. Pull total number of orders that were completed on 18th March 2023


Ans: SELECT COUNT(*) AS TotalCompletedOrders FROM Orders WHERE DATE(OrderDate) = '2023-03-18';


2. Pull total number of orders that were completed on 18th March 2023 with the first name ‘John’ and last name ‘Doe’


Ans: SELECT COUNT(*) AS TotalCompletedOrders FROM orders WHERE DATE(OrderDate) = '2023-03-18' AND FirstName = 'John' AND LastName = 'Doe';
 
3. Pull total number of customers that purchased in January 2023 and the average amount spend per customer


Ans: SELECT COUNT(DISTINCT CustomerID) AS TotalCustomers, AVG(Amount) AS AverageAmountSpentPerCustomer FROM Purchases WHERE YEAR(PurchaseDate) = 2023 AND MONTH(PurchaseDate) = 01;


4. Pull the departments that generated less than $600 in 2022


Ans: SELECT DepartmentName FROM DepartmentRevenue WHERE YEAR(RevenueDate) = 2022 GROUP BY DepartmentName HAVING SUM(Revenue) < 600;




5. What is the most and least revenue we have generated by an order


Ans: 
To find maximum revenue generated by an order
SELECT MAX(Revenue) AS MostRevenue FROM Orders;


To find minimum revenue generated by an order
SELECT MIN(Revenue) AS LeastRevenue FROM Orders;


6. What were the orders that were purchased in our most lucrative order


Ans: SELECT * FROM Orders WHERE Revenue = (SELECT MAX(Revenue) FROM Orders);
